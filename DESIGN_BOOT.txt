zebbs (zero'stage' embedded boot'loader' buddy system)

zebbs inits the devices and loads the febbs from BOOT.bin
- init device
- load uboot-spl into local ram
- if no uboot-spl, try to load app to ddr
- if uboot jump to local ram
- if app or no file loaded, jump to ddr

ubootspl/openSbi
- init ddr
- load opensbi to ddr
- load uboot to ddr
- jump to opensbi

opensbi
- init system
- jump to uboot

uboot
- load device tree into ddr
- load linux kernel into memroy
- start linux boot.
